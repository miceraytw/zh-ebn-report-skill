# 實證讀書報告模板（Evidence-Based Reading Report Template）

適用：N1、N2 進階報告、台灣實證護理學會投稿讀書報告類。

**與案例分析最大的差別：沒有真實個案，不跑 Application 與 Audit 兩個步驟**。

---

## 報告完整骨架

```
【篇名】       （建議疑問句；pipeline 以 refined_topic_zh_question 為 DOCX 標題）
【摘要】       （約 200 字）
【關鍵詞】

一、前言（臨床問題背景）
二、實證主題設定（PICO）
三、文獻搜尋策略
四、文獻評讀結果
五、文獻綜整與討論
六、臨床應用建議
七、結論
八、參考文獻（由 Quarto/CSL 自動產出）
```

Pipeline 側對應章節名（`section_names` / `SectionName`）：
`摘要 → 前言 → 主題設定 → 搜尋策略 → 評讀結果 → 綜整 → 應用建議 → 結論`。
字數常數集中於 `src/zh_ebn_report/spec/reading_report_spec.py`，所有 prompt 與 compliance 檢核均以此為 single source of truth。

## 各段寫作要點

### 篇名

建議用**疑問句式**呈現 PICO，讓審查者一眼看懂主題範圍：

**範例**：
- 「高密度泡棉床墊與一般床墊相比，在壓瘡高危險群病人之預防成效」
- 「腰椎穿刺後臥床平躺是否能預防頭痛發生率？」
- 「使用含 Glutamine 漱口是否可有效降低白血病病人口腔黏膜炎嚴重度？」
- 「全髖關節置換術後病人使用間歇充氣加壓器是否有效降低靜脈血栓栓塞發生率？」

### 前言

500–800 字，交代三件事：

1. **這個問題在臨床有多常見、多重要**（引用流行病學數據）
2. **現行處置的爭議或不確定性**（為什麼要做實證）
3. **本次報告的目的**

**範例開頭**：
> 「壓瘡在急、慢性醫療機構中是普遍且備受關注的健康議題。一旦發生壓瘡，對於傷口後續照護所花費的人力、物力與金錢，為現代健保應考量之成本效益。而壓瘡發生率更是護理品質照護的重要指標項目之一⋯⋯本實證讀書報告之目的，期藉由實證護理手法，針對臨床使用的減壓床墊進行文獻探討⋯⋯」

### 實證主題設定（PICO）

格式同案例分析（見 `case-report-template.md`）。讀書報告通常只設定 **1 個 PICO**。

### 文獻搜尋策略

**表格呈現**（同案例分析 `表一` 格式）：
- 資料庫：PubMed、Cochrane Library、CINAHL、Embase、華藝線上圖書館、台灣博碩士論文知識加值系統
- 搜尋條件：MeSH term、布林邏輯、Limits（年份、語言、研究設計）
- 搜尋歷程：起始篇數 → 排除條件 → 納入篇數

**範例敘述**：
> 「以 PICO 結構設定搜尋策略，以中文及醫學名詞關鍵字為：減壓床墊（foam bed）、床墊（bed）、壓瘡（pressure ulcer），搜尋公共醫學資料庫選用 PubMed 及 Cochrane Library 實證資料庫。設定 Limits：2015–2025、Human、Systematic Review、Meta-Analysis、RCT。」

**搜尋流程圖**（可選，但加分）：用 PRISMA 簡化版流程圖呈現「識別 → 篩選 → 納入」。

### 文獻評讀結果

讀書報告通常納入 **2–5 篇文獻**。每篇的評讀格式同案例分析（CASP + 證據等級）。

**摘要表**（放在評讀前）：

| 文獻 | 研究設計 | 證據等級 | 主要結果 |
|---|---|---|---|
| 文獻 1 | Meta-analysis | Level Ia | ⋯⋯ |
| 文獻 2 | RCT | Level Ib | ⋯⋯ |

### 文獻綜整與討論

500–800 字。**不是摘要的再摘要**，而是**比較與整合**：

1. **哪些結論一致？**（強證據方向）
2. **哪些結論不一致？**（需要進一步研究）
3. **研究設計上的差異如何影響結論？**
4. **台灣的醫療脈絡與這些研究有什麼差異？**

### 臨床應用建議

讀書報告沒有真實個案，但仍要寫**假想的應用情境**：

1. **目標族群**（什麼樣的病人適用）
2. **建議的護理措施**（條列 3–5 項）
3. **執行注意事項**
4. **成效評值指標**（若未來要執行，會用什麼量表或客觀指標評估）

**範例**：
> 「綜整兩篇證據等級分別為 Ia 及 Ib 的系統性文獻回顧及統合分析，針對壓瘡風險較高者，若可以使用高密度泡棉床墊，可以緩衝身體的表面壓力，支撐度優於一般床墊，可降低褥瘡的發生率。建議本單位針對 Braden scale 評分 ≤12 分之高危險群病人，提供高密度泡棉床墊；並搭配每 2–4 小時翻身之護理常規，成效指標為壓瘡發生率。」

### 結論

200–300 字。三句話可以寫完：
1. 本實證主題的明確結論
2. 推薦的臨床作法
3. 對未來研究或護理實務的建議

### 參考文獻

**至少 5 篇**，其中至少 2 篇高證據等級（SR、MA、RCT）。格式規範同案例分析（APA 7、中英分列）。

## 篇幅參考

| 段落 | 字數 / 頁數 | Pipeline 硬區間（CJK 字） |
|---|---|---|
| 摘要 | 約 200 字 | 180–260 |
| 前言 | 1 頁 | 500–800 |
| 主題設定（PICO） | 0.5 頁 | 200–400 |
| 搜尋策略 | 0.5–1 頁 | 500–800 |
| 評讀結果（2–5 篇） | 3–5 頁 | 500–1000 |
| 綜整討論 | 1–1.5 頁 | 500–800 |
| 應用建議 | 0.5–1 頁 | 300–600 |
| 結論 | 0.5 頁 | 200–300 |
| 參考文獻 | 1 頁 | ≥5 篇，其中 ≥2 篇 Level I–II |
| **總計** | **8–12 頁** | — |

> Pipeline 對字數採 **CJK 字數（Unicode 中日韓統一表意文字）** 統計，扣除 Markdown 表格列與 `[@citekey]` 引文佔位。LLM 自評的 `word_count_estimate` 僅供參考，以 `compliance.count_cjk_excluding_tables_and_cites()` 為準。

## 常見退稿原因（讀書報告類）

1. **題目太大**：「壓瘡的護理」這種範圍太廣，要縮為特定介入 vs 特定對照。
2. **文獻太舊**：5 年以上文獻比例過高（除非是關鍵經典研究）。
3. **只有摘要沒有評讀**：每篇都要有 CASP 表，不能只寫「本文結果顯示⋯⋯」就帶過。
4. **沒有納入/排除條件**：搜尋策略不透明。
5. **結論與證據不符**：例如只有 1 篇 RCT 就說「本實證建議全院改用 X」，強度不夠。
6. **用字太口語**：「我覺得⋯⋯」「好像⋯⋯」「差不多」等用語會被打回。
